Welcome to statFEM_analysis’s documentation

This is the documentation for code accompanying our error analysis of the Statistical Finite Element Method (StatFEM) 1. This code is hosted on GitHub here

Instructions for installing and running the code can be found in the “Getting started” section.

In the “Reference Documentation” section there are several notebooks which aim to explain in detail how the code was developed.

In the “Experiments from paper” section the scripts for the numerical experiments can be found. These scripts can be downloaded from here.
