Welcome to statFEM_analysis's documentation!
============================================

This is the documentation for code accompanying our error analysis of the Statistical Finite Element Method (StatFEM) :footcite:p:`papandreou2021theoretical`. This code is hosted on GitHub `here <https://github.com/YanniPapandreou/statFEM_analysis>`_ 

Instructions for installing and running the code can be found in the ":doc:`README`" section.

In the ":doc:`Notebooks`" section there are several notebooks which aim to explain in detail how the code was developed.

In the ":doc:`scripts`" section the scripts for the numerical experiments can be found. These scripts can be downloaded from `here <https://github.com/YanniPapandreou/statFEM_analysis/tree/main/scripts>`_.
